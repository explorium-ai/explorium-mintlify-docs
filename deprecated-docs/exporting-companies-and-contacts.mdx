---
title: "ExactInsights: Export Companies and Contacts to Salesforcece"
description: "In order to Export data from ExactInsights to Salesforce, First setup a valid Salesforce Integration and map relevant data points into Salesforce standard and custom fields."
---

<Steps>
    <Step>
    To Export data to Salesforce:

   1. Select relevant records to be exported in ExactInsights Companies or Contact Lists.
   2. Click "Export to CRM" and select the relevant Integration name. The Integration connection status must be "Valid" in order to select it for export.

   <Frame>
   ![Select an Integration to export data to](/images/deprecated-docs/5dee5fc-image.png)
   </Frame>

   Select an Integration to export data to
   </Step>

   <Step>
   Click "Export". While data is exported you will receive a notification.

   <Frame>
   ![](/images/deprecated-docs/a625118-image.png)
   </Frame>
   </Step>

   <Step>
   Any record that has been exported has a "CRM" indication with a timestamp of the latest export

   <Frame>
   ![](/images/deprecated-docs/6426e49-image.png)
   </Frame>

   </Step>
</Steps>
