Explorium's API provides access to the world's leading B2B data.

Use this API to combine business, contact, and geolocation data, and enrich up to 500 records every month for free!

## Where to begin

[Once you've gotten your API key](/reference/getting-your-api-key), browse the endpoints from the sidebar menu and pick the first data bundle you'd like to try.

<Frame>
  <img src="/images/deprecated-docs/353956b-Choose_Endpoint.png" />
</Frame>

Review the data signals included in the bundle by clicking on the data dictionary button at the bottom of the page

<Frame>
  <img src="/images/deprecated-docs/461c9a0-Click_for_dictionary.png" />
</Frame>

You'll find all data outputs' descriptions and formats:

<Frame>
  <img src="/images/deprecated-docs/c5659a0-Screen_Shot_2022-12-11_at_10.58.03.png" />
</Frame>

## Creating a request

On the right side of the screen, you'll quickly recognize the sample request pre-populated.

You can review the sample response to the request by clicking on the response button to the right -

<Frame>
  <img src="/images/deprecated-docs/477d862-Click_for_sample_response.png" />
</Frame>

Behold!

<Frame>
  <img src="/images/deprecated-docs/ca44638-Open_sample_response.png" />
</Frame>

You can make requests in the documentation center by updating your API key in the header, and edit the body parameters to include the inputs you're interested in enriching.

<Frame caption="updating body params for your request">
  <img src="/images/deprecated-docs/8c2c45a-Changing_request_params.png" />
</Frame>

The request window will update accordingly, the final step is to click on **Try It!** and see the response.

Let's get you started!
