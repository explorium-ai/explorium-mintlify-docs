title: "Data Catalog"

This table provides a comprehensive overview of the available data assets in the **Explorium AgentSource**. Each row represents a specific data field, categorized by event type and data asset, with detailed information about its purpose, structure, and accessibility.

For each field, you’ll find:

- **Display Name** – the user-friendly label shown in the UI
- **Name** – the internal identifier used in the system
- **Description** – explanation of the field's meaning and context
- **Ontology** – the data format (e.g., TEXT, DATETIME, BOOLEAN)

This catalog helps users understand what data is available, how to use it, and where it fits within the partner service infrastructure.
