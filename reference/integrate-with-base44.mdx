---
title: "Integrate with Base44"
description: "Add company & prospect data to any Base44 AI app in under 3 minutes."
---

## 1. Open the Base44 Dashboard

Launch [https://www.base44.com](https://www.base44.com/) and sign in.
You'll land on the "What would you build today?" home screen.

<Frame>
![](/images/reference/b19a7d9f910251b57c5f29c7f51d1759ad310de9e00328b8985209171a789d7f-Screenshot_2025-04-29_at_16.38.12.png)
</Frame>

## 2. Browse to Integrations <Icon icon="arrow-right" /> Catalog

Click the puzzle-piece icon in the left nav to open Integrations.
Locate Explorium in the Community Integrations list (you can use the search box).

<Tip>
**Tip:**

If you don't see Explorium yet, hit Create Integration and paste the link your CSM provided.
</Tip>

<Frame>
![](/images/reference/14cbaaae11b48ef67b3427b7085f9293bfb9d8c5b3861ccd5ee423276afd41a7-Screenshot_2025-04-29_at_16.38.21.png)
</Frame>

## 3. Review the Explorium Card

Click Explorium to open the integration detail page.
You'll see example prompts and a status badge of Approved.

Hit Use this integration to drop it into your new (or existing) project.

<Frame>
![](/images/reference/039c3986d9010662d374b40a077fa69dd86f0ee2e6bde1fec9882d08e4a18624-Screenshot_2025-04-29_at_16.38.30.png)
</Frame>

## 4. Provide Your EXPLORIUM\_API\_KEY

Base44 will prompt for required credentials.

Select New value (or “Reuse existing” if you’ve added it before).<br/>
Paste your **EXPLORIUM\_API\_KEY**.<br/>
You can grab a fresh key anytime from [https://admin.explorium.ai](https://app.explorium.ai/api-usage).<br/>
Once adding an api key – the red “**Missing required keys**” warning disappears.<br/>
That’s it! The Explorium data functions are now available in your Base44 backend cloud functions.
