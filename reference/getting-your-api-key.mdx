---
title: "Getting your API key"
---

When you sign up to Explorium's free platform trial you'll be assigned your unique API key. If you don't have a user yet, [sign up for free to create one](https://www.explorium.ai/signup/).

Once you've signed up and logged in, navigate to the API page through the link in the sidebar navigation.

<Frame>
![](/images/reference/5777af4-Home_screen1.png)
</Frame>

You'll find your API key and an updated view of your monthly consumption on this page.

Every Explorium user gets 500 record enrichments regardless of the number of requests made.

<Frame>
![](/images/reference/146c479-API_Liber.png)
</Frame>

Your API quota is reset at the beginning of every month, unused credits do not carry over so make full use of your trial!

Once you get your API key, browse the documentation and create your requests for the bundle of your choice.
