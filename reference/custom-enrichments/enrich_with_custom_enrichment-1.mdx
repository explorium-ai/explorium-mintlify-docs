## Consuming Custom Enrichment with the AgentSource API

After creating your custom enrichment, you can consume it through the AgentSource API. Here's how:

### Endpoint: `POST api.explorium.ai/v1/custom_enrichments/<custom_enrichment_id>/enrich`

## Description

Consume a custom enrichment by enriching a single business.

#### Parameters

- **data\_context**: DataFrame DICT - Rows of data that will be used for evaluation. Must include a business\_id field and can optionally include other user features.

#### Request Example

```shell Bash
POST api.explorium.ai/v1/custom_enrichments/<custom_enrichment_id>/enrich
```
